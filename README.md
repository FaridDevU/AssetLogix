# AssetLogix

**AssetLogix** is an internal web tool for managing documents, equipment, and projects within a company. It allows you to view, update, and track deliveries and returns of equipment and related documentation.

  Features
- Management of equipment assigned to projects.
- Control of delivery and return dates.
- Batch updates of columns and database schemas.
- Automated scripts for migrations and role setup.
- UI configured with Tailwind and Vite.

 Technologies Used
- Node.js  
- TypeScript  
- Vite  
- TailwindCSS  
- Drizzle ORM  


 🚫 Security
Sensitive files have been removed from this repository (`.env`, `cookies.txt`, `login_response.txt`).  
To configure your local environment, create your own `.env.local` file with the necessary variables.
